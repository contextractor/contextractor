# Publish All

Run the publish command to release the updated docs across all distribution channels.

## Run `/publish:all`

Execute `/Users/miroslavsekera/r/contextractor/.claude/commands/publish/all.md`

This publishes to all channels (Apify actor, npm, PyPI, Docker) and verifies each artifact post-publish.

## Success Criteria

- All channels published successfully
- Post-publish verification passes for PyPI, npm, Docker, and Apify
- All version files in sync
